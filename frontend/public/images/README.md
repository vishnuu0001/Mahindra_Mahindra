# Images Folder

This folder contains images used in the Manufacturing Digital Cockpit application.

## Required Images

### Landing Page Images
1. **manufacturing-hero.jpg** - Hero section background image showing modern manufacturing facility
2. **tech-mahindra-logo.png** - Official Tech Mahindra logo
3. **industry-4.0.jpg** - Industry 4.0 concept image
4. **digital-factory.jpg** - Smart factory/connected operations image

### Dashboard Images
5. **analytics-dashboard.jpg** - Analytics and data visualization
6. **smart-manufacturing.jpg** - Smart manufacturing operations
7. **iot-sensors.jpg** - IoT sensors in manufacturing

## Image Specifications

- **Format**: JPG or PNG
- **Hero Images**: Recommended size 1920x1080px
- **Icons/Logos**: PNG with transparent background
- **Dashboard Images**: 800x600px or higher

## Image Sources

To add manufacturing images, you can:

1. **Stock Photo Sites** (with proper licensing):
   - Unsplash (https://unsplash.com) - Free high-quality images
   - Pexels (https://pexels.com) - Free stock photos
   - Pixabay (https://pixabay.com) - Free images

2. **Search Keywords**:
   - "smart factory"
   - "industry 4.0"
   - "manufacturing automation"
   - "industrial iot"
   - "digital manufacturing"
   - "factory workers with tablets"

3. **Tech Mahindra Brand Assets**:
   - Contact Tech Mahindra marketing team for official logo and brand images

## Usage

Images can be referenced in React components using:

```javascript
// Public folder images
<img src="/images/manufacturing-hero.jpg" alt="Description" />

// Or using require for imported images
import heroImage from '../images/manufacturing-hero.jpg';
<img src={heroImage} alt="Description" />
```
